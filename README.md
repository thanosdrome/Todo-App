# Simple Todo App

A minimal and user-friendly Todo application to help you manage your daily tasks efficiently. Add, edit, delete, and mark tasks as completed with ease.

## Features
- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Responsive design
